# BusinessAccountingApp (macOS)

Минималистичное macOS-приложение для учета бизнеса с логином и dashboard.

## Как запустить на Mac

1. Убедитесь, что установлен Xcode (или Command Line Tools с поддержкой SwiftUI/macOS SDK).
2. В корне проекта выполните:

```bash
swift run BusinessAccountingApp
```

Если окно не видно сразу, приложение теперь принудительно активируется и выводит окно на передний план.

## Демо-логины

- `owner` / `12345`
- `manager` / `biz2026`
